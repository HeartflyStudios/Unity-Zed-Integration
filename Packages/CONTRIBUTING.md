# Contributing
Thank you for your interest in contributing to my proejct <3

## Guidance
If you have forked the repo and you want to merge your changes just create a PR via GitHub and I will do my best to get your contribution merged as soon as possible.

If you have a suggestion for improvement or issue to raise but you don't want to fix/implement it or don't know how, just open an issue via GitHub. I will get back to you as soon as I can with either:
- An explanation as to why the issue is not likely to be fixed/implemented, or why it can't be.
- An ETA on when I think I can get the fix/implementation done.

For both contributing and raising issues, if I don't respond within a week send me an email (address can be found below).
